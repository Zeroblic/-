
# 核心前端模块设计

Auth（登录认证）
       ↓
Upload（上传模块）
       ↓
Manage（视频管理）
       ↓
Recommend（灵感推荐）
       ↓
Analytics（数据可视化）

# 1️⃣ 登录模块（Auth Module）

### 功能点

* 登录
* 注册（可选）
* 前端路由守卫
* Token 缓存（localStorage）

### 技术点

* 使用 Axios + 请求拦截器把 Token 自动加到请求头
* 使用 React Router v6 做权限路由
* Ant Design Form组件

---

# 2️⃣ 短视频上传模块（Upload Module）

这是短视频系统的核心之一。

### 页面功能

* 上传视频文件（Multer 对接）
* 视频封面自动获取/上传
* 视频标题、分类、标签输入
* 上传进度条显示
* 发布视频

### 技术点

* 使用 `<Upload />` 组件 + 自定义请求实现大文件上传
* 使用 Hooks 实时显示 progress
* 发布信息 POST 给后端

### 前端页面拆分

```
/pages/upload/
   ├── index.tsx          # 主上传页
   ├── VideoUpload.tsx    # 文件上传组件
   ├── VideoForm.tsx      # 表单信息（标题/分类）
```

### 状态管理

* 上传进度、视频信息临时存在 Zustand/Context

---

# 3️⃣ 视频管理模块（Manage Module）

属于后台管理功能，包括已上传视频的：

* 查看
* 搜索/筛选
* 删除
* 编辑标题/分类
* 分页

### 页面结构

```
/pages/manage/
   ├── VideoTable.tsx      # 数据表格
   ├── VideoEditModal.tsx  # 编辑弹窗
   ├── FilterBar.tsx       # 搜索/筛选栏
```

### 技术点

* AntD Table + 分页
* 对接后端的 RESTful API
* 点击编辑 → 弹出 Modal → 编辑后保存更新

---

# 4️⃣ 推荐模块（Recommend Module）

虽然这里不是抖音首页那样的“推荐算法”，但它实现的是创作灵感的视频瀑布流/推荐展示。

### 功能

* 展示推荐视频（网格/瀑布流布局）
* 视频播放组件（封面/悬停预览）
* 无限滚动加载

### 页面结构

```
/pages/recommend/
   ├── index.tsx
   ├── VideoCard.tsx
   ├── WaterfallList.tsx      # 瀑布流布局
```

### 技术点

* CSS Masonry / react-masonry-css
* IntersectionObserver 实现无限滚动加载
* 视频预览组件封装

---

# 5️⃣ 数据可视化模块（Analytics Module）

视频的播放量、点赞量、趋势分析等。

### 功能

* 视频播放趋势折线图
* 各分类/标签数据统计
* 用户上传量趋势
* 推荐视频热度榜

### 技术点

* ECharts（或 Ant Design Charts）
* 后端返回结构化数据（views, likes, trendData）

### 页面结构

```
/pages/analytics/
   ├── OverviewChart.tsx
   ├── TrendChart.tsx
   ├── DataCard.tsx
```
