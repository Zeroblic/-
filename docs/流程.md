登录/注册流程
前端发送请求（POST /auth/login）
              ↓
Express 收到请求（app.js）
              ↓
匹配路由（auth.routes.js）
              ↓
router.post("/login", login)
              ↓
调用控制器 login()（auth.controller.js）
              ↓
控制器查询数据库 mysql2
              ↓
查询 user 表
              ↓
把结果返回给控制器
              ↓
控制器返回 JSON 给前端

JWT 登录机制
前端登录
   ↓
后端生成 token
   ↓
后端返回给前端
   ↓
前端把 token 存 localStorage
   ↓
前端请求需要登录的接口 → 请求头加 token
   ↓
后端中间件 verifyToken 验证 token
   ↓
通过 → 控制器执行真正业务
